# Address Book Page
"Manages address book for customer."

![Address book page](Screenshots/Address%20Book%20Page%20-%20icon.png)

This page type lets customers manage and edit their billing and shipping addresses.

## Technical Information

### Category
`Commerce`

### Inherits
[FoundationPageData](Foundation%20Page%20Data.md)

### Restrictions
The Address book Page is not available in edit view by default. To enable it, go to Admin view > Content type > [Commerce] Checkout Page > Settings and select the **Available in edit view** check box.\n
**Note**: This page type focusses on technical functionality. Therefore, you cannot edit the properties of the Content tab.
